Image Search Abstraction Layer

Example Usage:

https://abstract-image-search-asell22.herokuapp.com/api/imagesearch/cats%20lol

This url searches for images related to the query 'cats lol'. 10 images are sent back in JSON format.

Example Usage:

https://abstract-image-search-asell22.herokuapp.com/api/imagesearch/cats%20lol?offset=20

This url has a query string that retrieves different images. You can type in a different integer value for offset.

Example Usage:

https://abstract-image-search-asell22.herokuapp.com/api/latest

This url browses the most recent search queries.
